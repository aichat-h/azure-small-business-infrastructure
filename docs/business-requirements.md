# Azure Small Business Infrastructure Project

## Overview
This project simulates the design of a cloud infrastructure for a small company migrating from on-premises systems to Microsoft Azure.

The goal is to demonstrate understanding of core Azure services as part of preparation for AZ-900 and AZ-104 certifications.

---

## Business Scenario

A company called **TechSolutions Ltd.** is moving its IT systems to the cloud. The company has approximately 50 employees and requires a secure and scalable environment for hosting applications, storing files, and managing users.

Employees work both in-office and remotely.

---

## Requirements

The cloud environment must support:

- Secure hosting of internal applications
- Storage for company documents and files
- Secure remote access for employees
- Identity and access management for users
- Monitoring and basic backup capabilities
- A scalable and cost-effective architecture

---

## Goals of This Project

- Understand core Microsoft Azure services
- Practice cloud architecture design
- Build a foundation for future Infrastructure as Code (Bicep)
- Create a portfolio project for job applications
